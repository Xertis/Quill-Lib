### Mesh
Система Mesh доступна только на серверной стороне на текущий момент

Благодаря ней можно созздавать/управлять мешами, при этом меши сами синхронизируются с клиентами, самому синхронизировать их не надо

---
```lua
local mesh = require "meshup:api/api".server.mesh
```
- Импорт


```lua
mesh.to_mesh(
    pos1: vec3,
    pos2: vec3,
    destroy: bool
) -> Mesh
```
- **pos1** и **pos2** это крайние точки региона, который будет превращён в меш
- если **destroy** = **true**, то регион уничтожит те блоки, которые входят в его регион из сетки блоков
- Возвращает объект класса **Mesh**
```lua
mesh.to_struct(
    pos: vec3,
    mesh: Mesh
)
```
- Поворот меша должен быть кратен 90 градусам
- Устанавливает меш в сетку блоков на указанной позиции

### Объект Mesh

#### Свойства
Менять значения свойств напрямую ЗАПРЕЩЕНО, ФУ, НЕЛЬЗЯ, ОПАСНО
```lua
Mesh.origin: vec3 -- Позиция меша
Mesh.size: vec3 -- Размеры меша
Mesh.rotation: vec3 -- Вращение меша
```

#### Методы
```lua
Mesh:animation_play(animation_name: string) -- Запускает проигрывание анимации

Mesh:change_origin(pos: vec3) -- Меняет привязку меша
Mesh:set_pos(pos: vec3) -- Устанавливает позицию меша
Mesh:move(dir: vec3) -- Двигает позицию меша

Mesh:set_rot(rot: vec3) -- Устанавливает вращение меша
```
